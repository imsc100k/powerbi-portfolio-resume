# Power BI Resume Dashboard

An interactive Power BI resume designed to present professional experience, skills, and portfolio projects in a visually compelling, data-driven format.

This dashboard transforms a traditional resume into an analytical storytelling experience using Power BI.

---

## Overview

This Power BI Resume Dashboard highlights:

- Professional experience timeline
- Core technical and business skills
- Certifications and education
- Language proficiency
- Interactive portfolio previews
- Clean, executive-style design optimized for recruiters

---

## Tools & Technologies

- Power BI Desktop
- DAX
- Power BI Data Modeling
- Custom Visual Design
- Resume Data Structuring
- UI/UX for Dashboards

---

## Key Features

- Interactive experience timeline
- Skill radar and progress visuals
- Portfolio project previews
- Recruiter-friendly layout
- Optimized for quick decision-making
